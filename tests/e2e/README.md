# E2E тесты

Эта директория содержит сквозные (end-to-end) тесты для сервисов бэкенда:
- Сервис аутентификации (регистрация, вход, обновление токена, выход, информация о пользователе)
- Сервис пользовательского контента (закладки, отзывы, оценки)
- Async-API сервис (жанры, персоны, кинопроизведения)

## Требования

1. Go 1.22 или выше
2. Запущенные сервисы бэкенда
3. Базы данных заполненые dump.json 

## Запуск тестов

### Запуск всех тестов
```bash
go test -v
```

### Запуск тестов для конкретного сервиса
```bash
# Тесты аутентификации
go test -v -run TestAuth

# Тесты UGC
go test -v -run TestUGC

# Тесты Async-API
go test -v -run TestAsyncAPI
```

### Запуск отдельных тестов
```bash
go test -v -run TestAuthSignup
go test -v -run TestUGCGetBookmarks
go test -v -run TestAsyncAPIHealth
```

## Структура тестов

- `auth_test.go`: Тесты для эндпоинтов аутентификации
- `ugc_test.go`: Тесты для эндпоинтов UGC (закладки, отзывы, оценки)
- `async_api_test.go`: Тесты для эндпоинтов каталога фильмов

## Примечания

- Тесты автоматически пропускаются, если сервис недоступен
- Каждый тест создаёт собственные тестовые данные (пользователей, закладки и т.д.)
- Токены авторизации для UGC тестов получаются автоматически
